Main Screen

The Main Screen is where CSV files can be imported and specific
quarters (or the entire year) can be selected for report generation.
From this screen, rules cannot be created. However, clicking on
the "Rules" button will pop up a screen for modifying rules.

Importing CSV files

To import .csv files. click the browse button on the right side of
the panel. A screen will pop up prompting you to browse for a file.
Find a .csv file and click open. The pop up will close and the text
field will then show the absolute path of the selected file.
You can now press the "Import" button to import the data into
memory. This may take a few minutes to complete. It is
also possible to import a .csv file if you already know its
absolute pathname: just enter it into the text field under "CSV file".
After importing the .csv file, the list labeled "Municipality (County)"
will be populated with the Municipality/County combinations found
in the .csv file.

Specifying Reports

To create specify what kid of reports you wish to generate, press the
"reports" button to bring up a new pop up. The help page on the Reports
window explains creating custom reports in further detail.

Selecting a Quarter (or Entire Year)

To create a report for a specific quarter, just select the appropriate
radio button. Once the quarter is selected, any others will be deselected.
Now, when the "Generate Reports" is clicked, the reports will be generated
for the selected quarter or the entire year if selected.
